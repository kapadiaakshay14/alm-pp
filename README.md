# Advanced ALM for Power Platform

This repository holds the Azure DevOps pipeline templates, definitions and custom PowerShell scripts associated with the Advanced ALM for Power Platform course delivery by [Sean Astrakhan](https://www.linkedin.com/in/sean-astrakhan/) and [Parvez Ghumra](https://www.linkedin.com/in/parvezghumra/). Don't hesitate to reach out to us via the Heartbeat app in case of any queries.

We also deliver a shorter version of this course as an all-day hands-on workshop to delegates at major conferences around the world. This provides an interactive learning experience, with dedicated access to environments, Azure DevOps/Github organizations, source control repositories, agents/runners etc, as well as plenty of opportunities to ask questions. So far it has been delivered at the following events and is planned to be delivered at more events over the course of the coming months. If you would like us to bring this workshop to your event, or would like to be participate in one of these feel free to reach out.

[Canadian Power Platform Summit 2026: 20th-21st March, Vancouver, Canada](https://canadianpowerplatformsummit.com/workshops.html)
